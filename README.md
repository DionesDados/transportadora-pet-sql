# 🚚 Sistema de Banco de Dados - Transportadora Pet

## 📌 Sobre o projeto

Este projeto foi desenvolvido para demonstrar conhecimentos em Banco de Dados utilizando SQL.

O sistema simula uma empresa de transporte de produtos pet, permitindo o gerenciamento de:

- Motoristas
- Veículos
- Clientes
- Produtos
- Rotas
- Notas Fiscais
- Custos de Viagem
- Tempo de Entrega
- Manutenções
- Pagamentos

---

## 🛠 Tecnologias utilizadas

- SQL
- Banco de Dados Relacional
- Modelagem Relacional

---

## 📂 Estrutura do Projeto

- Criação das tabelas
- Inserção de dados
- Consultas SQL
- Relacionamentos entre tabelas
