# 🚚 Sistema de Banco de Dados - Transportadora Pet

Projeto desenvolvido para demonstrar conhecimentos em **Modelagem de Banco de Dados**, **SQL** e **Banco de Dados Relacional**, simulando o gerenciamento de uma empresa de transporte de produtos pet.

---

## 📖 Sobre o Projeto

O sistema representa uma transportadora responsável pelo transporte de produtos para lojas e distribuidores do segmento pet.

O banco de dados foi modelado utilizando relacionamentos entre tabelas, garantindo a integridade das informações através de **Primary Keys (PK)** e **Foreign Keys (FK)**.

---

## 🎯 Objetivos

- Criar um banco de dados relacional
- Aplicar conceitos de normalização
- Implementar relacionamentos entre tabelas
- Realizar consultas SQL
- Demonstrar conhecimentos em SQL para portfólio
